# Security Policy

## Scope

This repository contains markdown skill files (SKILL.md) — instructional text that Claude reads and follows. There is no executable code, no dependencies, and no build pipeline.

## What to Report

- A skill that instructs Claude to do something dangerous (delete files without confirmation, access sensitive directories, expose private data)
- A skill description that could be exploited for prompt injection
- Any content that could cause Claude to bypass its safety guidelines

## How to Report

Email security concerns to **milind@everydayaiapp.com** with:
- Which skill is affected
- What the risk is
- How it could be exploited

We will respond within 48 hours and fix confirmed issues within 7 days.

## What's NOT a Security Issue

- Skill output quality (use a bug report instead)
- Feature requests
- General questions about Claude or Cowork
