# Start Here

## Where are you on the Cowork journey?

Pick the one that sounds most like you:

---

### "I've never used Cowork" or "I tried it once and went back to Chat"

**You're at the beginning. That's the best place to be.**

Do this:
1. Run `/safe-first-task` — A 3-minute, zero-risk first experience. Nothing gets deleted, nothing gets modified. You just see what Cowork can do.
2. Run `/what-can-cowork-do` — Tell it your role and daily tasks. Get a personalized map of which Cowork features save you the most time.

**Time investment:** 10 minutes
**What changes:** You'll understand why Cowork isn't "ChatGPT with extra steps" — it's a fundamentally different way to work.

---

### "I use Cowork but the results feel generic" or "It doesn't know me"

**You're past the starting line but missing the setup that makes everything better.**

Do this:
1. Run `/cowork-setup-wizard` — 15 minutes to teach Cowork your role, preferences, and workflow. This single setup transforms every future session.
2. Run `/teach-claude-your-voice` — Share your writing style so output sounds like you, not like any AI.
3. Run `/delegation-coach` — Learn the 3-question framework that turns vague requests into great results.

**Time investment:** 30 minutes
**What changes:** Cowork stops producing generic output. Every session starts with your context already loaded. Your results go from "okay" to "I can actually use this."

### "I use Cowork regularly but feel like I'm only using 30% of it"

**You're productive. Now it's time to build systems.**

Do this:
1. Run `/cowork-health-check` — Get a score on your current setup and see what's missing
2. Run `/memory-system` — Set up persistent context so nothing is lost between sessions
3. Run `/workflow-builder` — Turn your repeating tasks into automated workflows
4. Run `/skill-customizer` — Tune any skill or workflow to YOUR specific needs

**Time investment:** 45 minutes
**What changes:** You move from "doing tasks" to "running systems." Consistent quality, less repetition, more time for the work that actually matters.

### "I want to unlock scheduling, phone control, and advanced features"

**You're ready for the features that make Cowork feel like the future.**

**What changes:** Work happens while you're away. Tasks run automatically. You text from your phone and come back to finished deliverables.
