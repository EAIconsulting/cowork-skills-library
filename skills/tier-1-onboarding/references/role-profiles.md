# Role Profiles — Tier 1 Onboarding Reference

Shared reference for all Tier 1 onboarding skills. Defines role-specific content so each
skill produces consistent, accurate guidance without independently inventing role context.

---

## How to Use This File

When a Tier 1 skill asks the user for their role, look up that role below and use the
Top 5 Use Cases, Best First Skill, and Highest-Impact Connected Tools to drive personalized
recommendations. Do not invent role-specific content — use what is here.

---

## 1. Marketing

**Top 5 Cowork Use Cases:**
1. Content pipeline automation — research, draft, optimize, and distribute in one run
2. Brand voice consistency — voice profile enforces tone across all content output
3. Campaign brief creation — structured briefs as files, ready to share with the team
4. Competitive research synthesis — pull multiple sources into a single organized summary
5. Social post batching — generate a week of posts from one long-form piece

**Best First Skill:** `/safe-first-task` → ask for a content pipeline for one topic

**Highest-Impact Connected Tools:** Google Drive (content storage), Slack (publishing
coordination), Google Calendar (campaign scheduling)

---

## 2. Consulting / Professional Services

**Top 5 Cowork Use Cases:**
1. Proposal and deck drafting — client-specific context via Projects keeps tone accurate
2. Meeting prep automation — agenda, attendee research, talking points as a pre-meeting file
3. Client status report scheduling — weekly reports run automatically, ready to send
4. Engagement documentation — deliverable templates formatted consistently across clients
5. Research synthesis — condense lengthy reports and filings into a structured summary

**Best First Skill:** `/safe-first-task` → ask for a meeting prep file for an upcoming call

**Highest-Impact Connected Tools:** Google Calendar (meeting context), Notion (project docs),
Gmail (client communication)

---

## 3. Operations / Project Management

**Top 5 Cowork Use Cases:**
1. Weekly status report automation — scheduled task runs every Monday, zero manual effort
2. Meeting notes to action items — paste notes, get a structured action-item file
3. Process documentation — interview stakeholders or existing notes → formatted SOP
4. Cross-tool dashboard — Slack + Calendar + email summarized in one daily briefing file
5. Risk and dependency tracking — structured documents updated on a recurring schedule

**Best First Skill:** `/first-scheduled-task` → automate the weekly status report

**Highest-Impact Connected Tools:** Slack (team updates), Google Calendar (sprint/project
milestones), Notion (documentation hub)

---

## 4. Finance / Analysis

**Top 5 Cowork Use Cases:**
1. Multi-document synthesis — read reports, spreadsheets, and filings in parallel; one output
2. Recurring summary reports — scheduled briefings from static data sources
3. Pre-meeting briefing prep — financial context, key metrics, and risk flags before calls
4. Structured deliverable templates — consistent formatting across all analyst output
5. Variance commentary drafting — paste numbers, get narrative explanations ready to edit

**Best First Skill:** `/safe-first-task` → ask for a briefing document from a recent report

**Highest-Impact Connected Tools:** Google Drive (data files), Gmail (stakeholder
communication), Notion (reporting workspace)

---

## 5. Sales / Business Development

**Top 5 Cowork Use Cases:**
1. Pre-call research and talking points — prospect-specific prep file before every call
2. Proposal drafting — structured proposal output using client context from CRM notes
3. Follow-up email drafting — personalized follow-ups based on call notes
4. Pipeline summary documents — deal-stage overviews formatted for leadership review
5. Outbound sequence drafting — multi-touch sequences written in the rep's voice profile

**Best First Skill:** `/safe-first-task` → ask for a pre-call research doc for a real prospect

**Highest-Impact Connected Tools:** Gmail (outreach and follow-up), Google Calendar
(call scheduling), Slack (team pipeline updates)

## 6. Executive / Leadership

**Top 5 Cowork Use Cases:**
1. Daily briefing — Slack, email, and calendar summarized into one morning file
2. Board and leadership deck drafting — structured decks from bullet inputs
3. Delegation templates — reusable task briefs so reports can run Cowork consistently
4. Strategic document drafting — memos, briefs, and OKR updates in consistent format
5. Inbox triage and response drafting — priority flagging and draft replies to clear backlog

**Best First Skill:** `/first-scheduled-task` → set up a daily briefing that runs automatically

**Highest-Impact Connected Tools:** Gmail (inbox), Slack (team pulse), Google Calendar
(schedule awareness)

**Top 5 Cowork Use Cases:**
1. Creative brief creation — structured briefs from loose client input, ready for team review
2. Client feedback synthesis — consolidate revision notes into a clear, prioritized action list
3. Content calendar planning — month-level planning documents from campaign goals
4. Research and inspiration gathering — organize reference material into themed summary files
5. Copy variations — generate multiple headline or caption options from a single brief

**Best First Skill:** `/safe-first-task` → ask for a creative brief for a current project

**Highest-Impact Connected Tools:** Google Drive (asset storage), Notion (project

## 8. Research / Analysis (Non-Finance)

**Top 5 Cowork Use Cases:**
1. Literature and document synthesis — process multiple sources in parallel, one summary file
2. Structured research reports — sections, citations, and key findings as an organized document
3. Data annotation and tagging — apply a consistent taxonomy to a batch of text inputs
4. Competitive landscape mapping — multi-source comparison table saved as a reference file
5. Recurring research digests — scheduled tasks that summarize new information on a topic

**Best First Skill:** `/safe-first-task` → ask for a synthesis document from recent reading

**Highest-Impact Connected Tools:** Google Drive (source documents), Notion (research
database), Slack (team knowledge sharing)

## Roles That Don't Fit the Above Categories

If the user's role doesn't map cleanly to one of these eight profiles, ask what percentage
of their week is spent on:

1. Writing and communication
2. Research and analysis
3. Recurring routine tasks
4. Coordination and project management

Use the highest-percentage category to select the closest matching profile above. For example,
a customer success manager who spends 60% on communication maps closest to Sales; a policy
analyst who spends 70% on research maps closest to Research / Analysis.
